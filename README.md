GEMM矩阵乘法设计

### 1.1 规格说明

●

实现高性能矩阵乘法硬件单元，算力1T OPS @FP16

●

实现功能：D=A \* B +C；A为（m,k）矩阵，B为（k,n）矩阵，C和D为（m,n）矩阵；A，B，C，D均为FP16数据类型

●

使用axi4.0的master接口作为GEMM数据的D矩阵输出

●

使用axi4.0的slave接口作为GEMM数据的A和B矩阵输入

●

加分项一：优化输入输出的数据带宽，并有足够的数据支撑

●

加分项二：单位算力的功耗优化，并有比较完备的测试数据
