# 模块级详细规格（Module-Level Specs）

本目录补充各模块的详细规格模板内容，统一包含：
- Feature（功能特性）
- Interface（接口定义）
- Block Diagram（模块框图）
- Architecture Diagram（架构/数据路径图）
- Function（功能行为与时序）

## 模块清单
- `M0_top_level.md`：顶层集成模块
- `M1_csr.md`：CSR/寄存器模块
- `M2_scheduler.md`：调度控制模块
- `M3_dma_axi.md`：DMA/AXI 模块
- `M4_buffer.md`：片上缓存模块
- `M5_compute_core.md`：计算核心（PE 阵列）
