# S4 - 验证规格（Verification Spec）

## 1. 验证目标
- 功能正确性
- 接口协议正确性
- 异常处理正确性
- 覆盖率达标

## 2. 验证方法
- 参考模型对比（Python/Numpy）
- UVM/SV testbench（或等效）
- 约束随机 + 定向用例

## 3. 用例分层
- L0：基础算例（小矩阵）
- L1：随机维度/随机数据
- L2：边界与异常（维度1、非整 tile、非法参数）
- L3：压力测试（长 burst、backpressure）

## 4. 覆盖率指标（建议）
- 功能覆盖率 >= 95%
- 代码覆盖率 >= 90%（行/分支/FSM）
- 关键 cross coverage 全命中

## 5. 签核条件
- 回归全绿
- 无 P0/P1 未关闭缺陷
- 覆盖率达到门限
