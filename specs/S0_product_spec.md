# S0 - 产品与需求规格（Product Spec）

## 1. 范围
实现 FP16 GEMM 加速单元，计算公式：
\[
D = A \times B + C
\]

## 2. 数据与维度
- 数据类型：`A/B/C/D` 均为 FP16
- 维度约束：
  - `A(m,k)`
  - `B(k,n)`
  - `C(m,n)`
  - `D(m,n)`

## 3. 接口需求
- 输入通路：AXI4 Slave（配置 + 数据输入）
- 输出通路：AXI4 Master（D 写回）

## 4. 性能目标
- 峰值目标：1 TOPS @ FP16（理论峰值，按时钟频率与并行度计算）
- 支持带宽优化与 TOPS/W 优化

## 5. 非功能需求
- 参数可配置（m/n/k、tile、stride、地址）
- 任务可观测（busy/done/err）
- 错误可诊断（非法参数、越界、AXI 错误）

## 6. 验收准则
- 功能正确：与参考模型一致
- 接口正确：AXI 协议合法
- 性能报告：吞吐、带宽利用率、功耗摘要
