# 01 - CSR 与寄存器映射

## 1. 模块职责
- 接收软件配置
- 控制任务启动/停止
- 上报状态与错误
- 提供中断能力

## 2. 推荐寄存器（示例）
| 偏移 | 名称 | 位段 | 描述 |

| 0x000 | CTRL | [0]start [1]soft_reset [2]irq_en | 控制寄存器 |
| 0x004 | STATUS | [0]busy [1]done [7:4]err_code | 状态寄存器 |
| 0x008 | DIM_M | [15:0] | M 尺寸 |
| 0x00C | DIM_N | [15:0] | N 尺寸 |
| 0x010 | DIM_K | [15:0] | K 尺寸 |
| 0x020 | ADDR_A_L | [31:0] | A 基地址低位 |
| 0x024 | ADDR_A_H | [31:0] | A 基地址高位 |
| 0x028 | ADDR_B_L | [31:0] | B 基地址低位 |
| 0x02C | ADDR_B_H | [31:0] | B 基地址高位 |
| 0x030 | ADDR_C_L | [31:0] | C 基地址低位 |
| 0x034 | ADDR_C_H | [31:0] | C 基地址高位 |

## 3. CSR 时序约定
- `start` 由 0->1 触发任务，硬件自动清零（建议）
- `done` 写 1 清除（W1C）
- `busy=1` 时修改关键寄存器应返回错误或忽略

## 4. 错误码建议
- `0x1` 参数非法（维度 0）
- `0x2` 地址未对齐
- `0x3` 地址越界
- `0x4` DMA 传输异常

## 5. 软件编程模型
1. 写入维度、地址、stride、tile
2. 清状态位
3. 置 `start`
4. 轮询 `done` 或等待中断
