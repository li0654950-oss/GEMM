# 05 - 计算核心（PE Array）

## 1. 模块职责
- 执行 tile 内矩阵乘累加
- 完成 `acc + C` 融合
- 输出 D tile 到写回路径

## 2. PE 阵列参数
- `PE_M`：行并行度
- `PE_N`：列并行度
- 推荐与 `TILE_M/TILE_N` 整除关系匹配

## 3. 运算数据路径
- 输入：FP16 `a`, FP16 `b`
- 乘法：`mul = a * b`
- 累加：`acc += mul`
- 输出：`d = acc + c`

## 4. 精度模式建议
- 模式 A：FP16 乘 + FP16 累加（资源省）
- 模式 B：FP16 乘 + FP32 累加 + FP16 输出（精度优）

## 5. 时钟与流水
- 乘法级与加法级可分级流水
- 使用 `valid/ready` 保证背压安全
- 对齐不同流水深度下的 C 路径延迟

## 6. 性能估算
- 每周期 MAC 数 ≈ `PE_M * PE_N`
- 理论 TOPS ≈ `2 * PE_M * PE_N * Freq`
